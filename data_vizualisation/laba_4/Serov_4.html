В качестве примера был создан дашборд для авиакомпании Invistico. Набор данных
был получен с платформы соревнований kaggle
(https://www.kaggle.com/sjleshrac/airlines-customer-satisfaction).
Дашборд был разбит на два основные листа:
 общая сводка по клиентам,
 использование клиентами компании и их оценки (фидбэк)
Готовый дашборд доступен по следующей ссылке:
https://lookerstudio.google.com/s/h29jhZ4luy4 .
Рисунок 1 – Общая сводка по клиентам
В качестве инструментов была использована пончиковая диаграмма с
распределением полов, столбчатая диаграмма, отражающая количество людей по возрастам.
Также была использована столбчатая диаграмма для распределения транзакций при оплате с
картой лояльности и без неё и сводная тепловая карта по распределению удовлетворения и
класса пассажира. Еще представлена сводка с текущим количеством клиентов, для понимания
объема выборки, что был принят для анализа.
Рисунок 2 – Использование клиентами компании и их оценки (фидбэк)
На рисунке 2 приведены следующие данные нормированная столбчатая диаграмма с
распределением оценок во время перелета и использования сервисов (wifi). Столбчатая
диаграмма c данными оценок по сервису бронирования. Есть таблица с оценками по онлайн
регистрации и пончиковая диаграмма с оценкой уровня комфорта. Еще представлена сводки с
текущими задержками для понимания, что может формировать данные оценки.
